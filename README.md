# 📘 Lista de Exercícios – Estruturas Condicionais em JavaScript

Este repositório contém uma coleção de **15 exercícios básicos em JavaScript**, com foco no uso de **estruturas condicionais** (`if`, `else if`, `else`).

O objetivo é praticar lógica de programação e tomada de decisão utilizando comparações e operadores lógicos.

---

## 🚀 Como Executar

Você pode executar os exercícios de três formas:

### ✅ 1. Usando Node.js
1. Instale o Node.js
2. Salve o arquivo como `exercicios.js`
3. No terminal, execute:

```bash
node exercicios.js
